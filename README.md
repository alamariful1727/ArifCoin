# ArifCoin

A simple blockchain which runs on PoW.

### Prerequisites

You should install `Node`, `git`

### Installing

`git clone https://github.com/alamariful1727/ArifCoin`

`cd ArifCoin`
`npm install`
`npm start`
For creating a node for the system `npm run n2`
